## Тема «Коллекции List»

### Условие задачи

#### ЗАДАЧА 1
Эту задачу следует выполнять в одном классе - классе Main.

Мы будем писать список покупок. Программа должна будет общаться с пользователем в следующем формате:

1. В цикле предлагать ему на выбор одну из операций: добавить, показать, удалить.
2. Пользователь вводит номер операции
3. Программа запрашивает дополнительные данные для совершения операции, эти данные вводятся одной строкой
4. Программа выполняет операцию, выводит информацию на экран и переходит обратно к пункту вывода меню

**Операция "добавить"**
Операция запрашивает какую покупку добавить в список, добавляет и выводит на экран информацию об итоговом количестве покупок.

**Операция "показать"**
Операция не запрашивает дополнительных данных и выводит все покупки с их нумерацией.

**Операция "удалить"**
Операция выводит на экран все покупки и предлагает ввести либо номер покупки для удаления, либо название самой покупки. После чего выводит еще раз список

#### ЗАДАЧА 2

**Операция "найти"**
Добавьте операцию поиска покупок по ключевым словам. При этом не должен учитываться регистр букв.
